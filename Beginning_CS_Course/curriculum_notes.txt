

Integer - whole numbers (e.g. 1, 2, 3, 4, 5)

Synonyms: int (C++, Java, C#)
Float - decimal numbers (e.g. 1.0, 2.5, 3.14)

Synonyms: double (C++, Java, C#)
Boolean - true or false values (e.g. True, False)

Synonyms: bool (C++, Java, C#)
String - sequence of characters (e.g. "hello", "world")

Synonyms: str (C++, Java, C#)
List - ordered collection of items (e.g. [1, 2, 3], ["apple", "banana", "cherry"])

Synonyms: array (C++, Java), list (Ruby)
